# Supervised Machine Learning Projects

This folder contains projects based on **Supervised Machine Learning algorithms**, where the model is trained using labeled data.

## Algorithms Used
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine
- K-Nearest Neighbors

## Tools & Libraries
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Objective
These projects demonstrate how supervised learning algorithms can be used for **classification and regression problems**.
